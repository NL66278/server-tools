LDAP users menu
===============

Provide quick access to LDAP settings under Users menu.

Usage
=====

Edit LDAP settings through Settings ==> Users ==> LDAP.

Bug Tracker

`GitHub Issues <https://github.com/OCA/server-tools/issues>`_.
